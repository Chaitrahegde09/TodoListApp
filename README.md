To-Do Android App

A clean and minimal Android To-Do application built using Kotlin and XML. The app allows users to add multiple tasks, manually mark tasks as completed with a single tap, and clear all tasks. Designed with a soft pastel UI for a pleasant user experience.

3. Clear all tasks easily

4. Simple, clean pastel UI

5. Beginner-friendly Android architecture

Tech Stack:

1. Language: Kotlin

2. UI: XML (LinearLayout, ListView)

3. IDE: Android Studio

Getting Started:

1. Clone the repository

2. Open in Android Studio

3. Sync Gradle files

4. Run on emulator or physical device
